## âœ¨ Key Features

- **Dual-Objective Core Loop:** A balancing act between a dynamic dirt-generation system (requiring constant cleaning to avoid game-over) and a high-stakes stealing mechanic.
- **Advanced Stealth AI:** Implemented varied guard behaviors (Patrol and Smart Guards) utilizing Unity's **NavMesh** navigation, vision cones, and dynamic alert logic that scales when thefts are detected.
- **Dynamic Lighting & 2.5D Rendering:** Utilizes Unity's **Light 2D** system and custom rendering layers to create an immersive, stylized museum atmosphere with camera detection zones.
- **Optimized Architecture:** Integrated a global Audio Pool for optimized sound effect playback, ensuring smooth performance without instantiation overhead.
- **UI Systems:** Dynamic health/dirt trackers, and an interconnected currency system.
